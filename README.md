# Pizza Sales Performance Report — Excel

## Overview
An interactive Excel dashboard analyzing 1,004 pizza orders across 6 restaurant chains, 
tracking delivery performance and payment behavior.

## Tools
Excel

## Key Metrics
- Total Orders: 1,004
- Average Estimated Duration: 11.87 min
- Average Delivery Duration: 29.49 min

## Features
- Slicers for Order Month, Payment Method, Restaurant Name, and Payment Category
- Charts comparing average delivery time by restaurant, order volume by category, and 
  payment method usage
- Delivery efficiency and distance analysis across dozens of cities
- Monthly order volume trends
